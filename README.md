# Currency Converter Web App

A lightweight and responsive currency converter application built using HTML, CSS, and JavaScript. It fetches real-time exchange rates using the Fetch API and allows users to convert between different currencies easily.

## Features
- Real-time currency conversion
- Fetch API integration
- Responsive UI
- Easy-to-use interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
